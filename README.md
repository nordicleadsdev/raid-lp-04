# RAID: Shadow Legends - Landing Page 04

Quiz-style prelander with glassmorphism design and particle effects.

## Overview
This is a multi-step questionnaire/funnel prelander for RAID: Shadow Legends, featuring:
- Age verification (18+)
- Geographic eligibility check
- Gameplay commitment questions
- Strategy preference selection
- Glassmorphism UI with backdrop blur
- Particle.js effects
- Left-aligned layout on desktop
- Responsive design

## Deployment
**Production URL:** https://us.play4free.net/04/

## Files
- `play.html` - Main quiz/funnel page (currently deployed)
- `index.html` - Alternative version
- `playnow.html` - Another variant
- `dramatic_quiz_version.html` - Dramatic styling variant
- `modified_quiz_left.html` - Left-aligned quiz variant
- `sea_conquest_quiz_replica.html` - Sea Conquest style replica

## Features
- Multi-step quiz flow
- Browser detection
- Glassmorphism design (backdrop-filter blur)
- Particle effects background
- Smooth transitions and animations
- Mobile responsive
- Left-aligned on desktop (>1200px)

## Technologies
- Vanilla JavaScript
- Particle.js
- CSS3 (Glassmorphism, Backdrop Filter)
- Google Fonts (Poppins)

## Setup

2. Uses external CDN for Particle.js
